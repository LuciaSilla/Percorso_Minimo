Prova prompt

Per permettere all'utente di inserire un numero dinamico di nodi e pesi, puoi aggiungere un pulsante o un link "Aggiungi nodo" al tuo form. Quando l'utente fa clic su questo pulsante, puoi creare dinamicamente nuovi elementi HTML per consentire l'inserimento dei valori. Di seguito è riportato un esempio di come è possibile implementare questa funzionalità: 1. Modifica il tuo form HTML aggiungendo un pulsante "Aggiungi nodo": ```html
``` 2. Modifica il tuo script JavaScript per gestire l'evento del pulsante "Aggiungi nodo" e creare dinamicamente nuovi elementi HTML per i nodi: ```javascript var nodoCounter = 1; // Contatore per i nodi document.querySelector('#aggiungi-nodo').addEventListener('click', function() { var nodiContainer = document.querySelector('#nodi-container'); // Crea un nuovo elemento HTML per un nodo var nodoDiv = document.createElement('div'); nodoDiv.classList.add('nodo'); var label = document.createElement('label'); label.for = 'nodo' + (nodoCounter + 1); label.textContent = 'Nodo ' + (nodoCounter + 1) + ':'; var input = document.createElement('input'); input.type = 'text'; input.classList.add('nodo-input'); input.required = true; nodoDiv.appendChild(label); nodoDiv.appendChild(input); nodiContainer.appendChild(nodoDiv); nodoCounter++; }); ``` 3. Modifica il tuo script JavaScript per recuperare i valori inseriti per tutti i nodi e salvarli come desideri: ```javascript document.querySelector('form').addEventListener('submit', function(event) { event.preventDefault(); // Evita il comportamento di default del form // Recupera i valori inseriti dall'utente per tutti i nodi var nodiInputs = document.querySelectorAll('.nodo-input'); var nodi = []; nodiInputs.forEach(function(input) { nodi.push(input.value); }); // Recupera gli altri valori inseriti dall'utente (come il peso) var peso = document.querySelector('#peso').value; // Salva i valori come desideri (ad esempio, in un array o in una variabile) var archi = { nodi: nodi, peso: peso }; // Fai qualcosa con i valori salvati (ad esempio, stampali in console) console.log(archi); // Successivamente, puoi eseguire altre operazioni o reimpostare il form document.querySelector('form').reset(); // Resetta i nodi creati dinamicamente e il contatore document.querySelector('#nodi-container').innerHTML = ''; nodoCounter = 1; }); ``` Con queste modifiche, ogni volta che l'utente fa clic sul pulsante "Aggiungi nodo", verrà generato un nuovo elemento HTML per consentire l'inserimento di un nuovo nodo. Alla fine, quando l'utente preme il pulsante "Invia", i valori di tutti i nodi e del peso verranno recuperati e salvati come desideri nel tuo script JavaScript.
